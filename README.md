# Medic Toro

Landing page genérica para una clínica veterinaria especializada en ganado y toros de lidia.

## Requisitos

- Node.js 20+
- npm 10+

## Instalación

```bash
npm install
```

## Scripts

| Comando           | Descripción                    |
|-------------------|--------------------------------|
| `npm run dev`     | Servidor de desarrollo         |
| `npm run build`   | Build de producción            |
| `npm run preview` | Preview del build              |
| `npm run test`    | Ejecutar pruebas unitarias     |
| `npm run lint`    | Linter (oxlint)                |

## Estructura

- `src/components/` — Secciones de la landing (header, hero, services, about, contact, footer)
- `src/data/` — Contenido estático (servicios, stats, campos del formulario)
- `src/pages/` — Página principal compuesta

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en el navegador.
